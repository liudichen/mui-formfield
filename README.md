# mui-formfield

用来准备用于formily的form项组件。

## Getting Started

Install dependencies,

```bash
$ npm i mui-formfield
```



Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father-build`,

```bash
$ npm run build
```
